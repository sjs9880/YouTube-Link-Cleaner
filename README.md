# YouTube Link Cleaner

유튜브 링크에서 불필요한 추적 코드를 자동으로 제거하고, 깔끔한 링크를 공유할 수 있도록 도와주는 크롬 확장 프로그램입니다.

## 주요 기능

*   **추적 코드 제거**: URL에서 `si`, `feature`, `pp`, `gclid` 등 불필요한 추적 파라미터를 자동으로 제거합니다.
*   **주소창 자동 정제**: 유튜브 페이지 진입 시 주소창의 URL을 즉시 깔끔하게 정리합니다.
*   **복사 시 링크 정제**: `Ctrl+C` 등으로 링크를 복사할 때, 추적 코드가 제거된 깨끗한 링크가 클립보드에 복사됩니다.
*   **Shorts 변환**: (옵션) `Shorts` 링크를 일반 동영상(`watch`) 링크로 자동 변환하여 복사할 수 있습니다.
*   **마크다운 복사**: (옵션) 복사 시 `[제목](URL)` 형태의 마크다운 포맷으로 복사할 수 있습니다.
*   **기타 제거 옵션**: 재생 위치(`t`), 재생 목록(`list`) 파라미터 제거 여부를 선택할 수 있습니다.

## 설치 방법

1.  이 저장소를 다운로드하거나 클론합니다.
2.  크롬 브라우저 주소창에 `chrome://extensions`를 입력하여 이동합니다.
3.  우측 상단의 **'개발자 모드'**를 켭니다.
4.  **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭하고, 다운로드한 폴더를 선택합니다.

## 설정

확장 프로그램 아이콘을 클릭하여 팝업 메뉴에서 다음 기능을 켜고  끌 수 있습니다.

*   **추적 코드 제거**: 기본 추적 파라미터 제거 기능 활성화/비활성화
*   **타임스탬프 제거**: URL의 시간 정보(`t=123s`) 제거
*   **재생목록 제거**: 재생목록 정보(`list=...`) 제거
*   **Shorts → 일반 영상 변환**: 쇼츠 링크를 일반 플레이어 링크로 변환
*   **마크다운 링크 복사**: 제목과 링크를 마크다운 형식으로 복사
